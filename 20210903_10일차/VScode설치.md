## VS설치 사이트
  https://code.visualstudio.com/
  
  
## 환경설정
  - ctrl + shift + x
  - korea 검색, 지구본 모양설치 : 한글화
  - Live Server 설치 : 오른쪽 클릭해서 바로 html 사이트에 띄울수있음
  - Beautify : 자동 정렬 가능.
    - ctrl + shift + p
    - 바로가기 검색
    - 바로가기 키에 들어가서
    - Beautify file 의 원하는 키 바인딩을 해주면됨 ( 교수님은 ctrl + l 로 하심)

I-ON Deploy Server

다량의 서비스 서버를 운영하고, 동시다발적으로 배포해야 할 다양한 콘텐츠 자산들을 보유하고 있는 기업을 위한 최적의 콘텐츠 배포 자동화 솔루션입니다.

제품소개 다운로드




분산 환경과 멀티 플랫폼 상에서 파일 전송/배포 및 서버간 동기화 기능을 모두 제공하는 FTP와 rSync의 Hybrid적인 콘텐츠 배포 자동화 솔루션입니다




개요

아이온커뮤니케이션즈의 IDS는 콘텐츠 배포 시 오류를 야기할 수 있는 기존의 수동 콘텐츠 배포 대신, 배포 프로세스의 자동화, 다수의 서버에 동시 디플로이, 스케쥴 배포 등 콘텐츠의 신속하고 정확한 업데이트를 위한 파워풀 한 기능을 제공하여, 기업 사이트의 운영에 효율성을 극대화합니다.




Architecture





모듈별 주요기능

업로드매니저

파일 서버상의 자원을 관리하기 위한 등록/삭제/승인과 같은 전반적인 관리 역할을 합니다.

디플로이매니저

업로드매니저로부터 동기화 된 배포 대상이 되는 파일의 관리 및 배포 작업 컨트롤,모니터링하는 역할을 합니다.

어드민매니저

파일서버&호스트 관리, 사용자 및 그룹관리, 승인 룰 설정, 트레이스 관리 등 IDS 시스템의 전반적인 관리 기능을 담당합니다.






특장점

파일 이관 관리
개발, 검증, 운영 서버간 파일 이관 시, 각 단계 이관 담당자의 승인 프로세스를 통해 Task 단위로 파일을 이관할 수 있으며, 그 절차도 설계할 수 있도록 지원

안전하고 빠른 파일배포
서버간 파일 이관 시 자체 프로토콜을 활용하여 128bit, 256bit 암호화 과정을 설정, 전송하므로 안정적으로 이관 가능. 중간에 파일을 탈취하기 어려우며, 탈취되더라도 확인이 불가하여 안전한 파일의 이관 진행

사이트 통합관리
일대다, 다대다 등 복수개의 서버를 운영 시, 서버 간의 파일 전송, 동기화 등 다양한 조건에 대하여 복수개의 서버간 파일 이관 지원




네트워크 사용량 감소
파일의 압축 배포를 통해 물리적인 네트워크 사용량이 감소

유지보수 비용절감
서비스에 반영하기 전 개발과정에서 프로그램의 치명적인 오류를 검출하여, 개발 이후 보완을 위한 유지보수 비용이 감소

업무효율성 증가
스케줄 자동배포와 Transaction 기반의 동보전송 지원으로 파일 배포시의 발생하는 작업 시간을 감축시킬 수 있습니다. 배포 히스토리 관리, 버전 별 롤백 기능 등으로 사이트 운영업무에 대한 제반 업무량 감소

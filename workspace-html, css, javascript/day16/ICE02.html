ICE Campaign

정확하고 효과적인 고객 데이터 관리를 위해 DW/CRM시스템과 CMS시스템을 연동하였습니다. 고객데이터를 관리하고 이를 바탕으로 성공적인 고객 맞춤형 마케팅을 지원하는 캠페인 솔루션입니다.

제품소개 다운로드



I-ON Contents Ecosystem Campaign은 DW(Date Warehouse)시스템으로 수집 된 고객 데이터를 사용한 맞춤형 마케팅 자동화 환경을 제공하며, CMS기반으로 손 쉽게 행사카드, 모바일DM 등의 제작이 가능합니다.

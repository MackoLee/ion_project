직관적인 UI

컨택쉐어는 사용자 중심의 미니얼 인터페이스를 제공합니다. 연락처를 공유하기 편하도록 간단한 메뉴로 구성되어 있습니다.

PC, 태블릿, 모바일 환경에서 지원

컨택쉐어는 PC, 스마트폰, 태블릿 PC 등 다양한 기기 유형을 지원합니다. 기기와 상관없이 자동으로 동기화된 최신 연락처를 보유할 수 있습니다.

연락처 권한 설정

상대방에 따라 공유할 연락처의 수정, 공유, 소유 권한을 개별로 지정할 수 있습니다.
